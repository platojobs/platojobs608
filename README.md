# 2022年

`gitbook` 生成电子书主要有三种方式:

* `gitbook-cli` 命令行操作，简洁高效，适合从事软件开发的相关人员.
* `gitbook-editor` 编辑器操作，可视化编辑，适合无编程经验的文学创作者.
* `gitbook.com` 官网操作，在线编辑实时发布，适合无本地环境且科学上网的体验者.
